## Cursor

- **Hobby (Free):** $0/month — limited completions, 50 slow premium requests
- **Pro:** $20/user/month (or $16/month billed annually) — unlimited Tab completions, $20/month usage credits for premium models, unlimited Auto mode
- **Pro+:** $60/user/month — 3x usage credits vs Pro
- **Ultra:** $200/user/month — 20x usage credits vs Pro
- **Business (Teams):** $40/user/month — Pro-equivalent AI + admin controls, centralized billing, shared team rules
- **Enterprise:** Custom pricing — contact sales

Source: https://cursor.com/pricing — verified 2026-05-12

> Note: Since June 2025, Cursor moved from request-based to credit-based billing. The $20 Pro plan includes $20/month in usage credits for frontier models; Auto mode is unlimited.

## GitHub Copilot

- **Free:** $0/month — 2,000 code completions/month, limited chat
- **Pro:** $10/user/month (or $100/year) — unlimited completions, 300 premium requests/month
- **Pro+:** $39/user/month — 1,500 premium requests/month, access to GPT-4.1 and additional models
- **Business:** $19/user/month — centralized billing, org policy controls, audit logs, SAML SSO, IP indemnity
- **Enterprise:** $39/user/month (requires GitHub Enterprise Cloud) — custom knowledge bases, fine-tuned models, advanced audit

Source: https://docs.github.com/en/copilot/concepts/billing/organizations-and-enterprises — verified 2026-05-12
Source: https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/ — verified 2026-05-12

> Note: GitHub is transitioning all Copilot plans to usage-based (token/credit) billing effective June 1, 2026. Seat prices remain unchanged through the transition.
